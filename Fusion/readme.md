1.ent_create创建的实体不能马上用ent_fire执行，需要分开；  
2.由于指令过多，alias语句不能全部丢到一个cfg加载，不然会跳过部分alias；  
3.轮盘的指令cl_radial_radio_tab_1_text_8 cmd";"这一部分尽量先定义别名，不然执行的第一个指令会给忽略掉。
